# AI

## Локальный запуск ИИ

Linux идеально подходит для запуска нейросетей локально благодаря эффективной работе с GPU и файловой системой.

## Ollama 

[Ollama](https://ollama.com/) cамый простой инструмент для запуска LLM (Large Language Models) в терминале Linux.

- Установка: <code>curl -fsSL https://ollama.com/install.sh | sh</code>.
- Перезапуск: <code>sudo systemctl restart ollama</code> 

Команды:

- <code>ollama ps</code>, показывает только те модели, которые запущены в данный момент и потребляют ресурсы (RAM/VRAM)
- <code>ollama list</code>, показывает все модели, которые вы когда-либо скачивали на свой компьютер.

> [!Note]
>  По умолчанию Ollama выгружает модель сама через 5 минут бездействия.


### DeepSeek-Coder-V2

Работа с DeepSeek-Coder-V2 через Ollama в Linux позволяет получить мощного помощника по коду, который сопоставим по
качеству с GPT-4 Turbo, но работает полностью локально.

- Запуск модели: <code>ollama run deepseek-coder-v2</code>.

По умолчанию загружается облегченная версия (Lite, 16B). Если у вас мощное оборудование (более 128 ГБ ОЗУ), 
можно попробовать полную версию :236b, но для большинства задач и видеокарт оптимальна именно Lite.

**Системные требования**

Минимум: 8 ГБ ОЗУ (для самых маленьких квантованных версий), но рекомендуется от 16 ГБ ОЗУ и видеокарта с 8–12 ГБ VRAM 
(серии RTX 3060 и выше) для комфортной скорости генерации.

GPU: Ollama на Linux автоматически подхватывает драйверы CUDA. Убедитесь, что они установлены, чтобы вычисления 
шли на видеокарте, а не на процессоре

### Qwen2.5-Coder

Qwen2.5-Coder (1.5B или 7B) — отличная современная альтернатива

- Запуск модели: <code>ollama run qwen2.5-coder:1.5b</code>, займет около 1.5 ГБ RAM
- Запуск модели: <code>ollama run qwen2.5-coder:7b</code>, займет около 4.7 ГБ RAM

> [!Note]
> Для комфортного программирования на Linux с локальным ИИ лучше всего подходит 
> Qwen2.5-Coder:7b — она быстрее DeepSeek на среднем железе и почти не уступает в качестве.

## Инструменты для многофайлового редактирования

[Aider](https://aider.chat/) [git](https://github.com/aider-ai/aider) - многофункциональный инструмент 
для редактирования текста, который подходит для работы с Qwen2.5-Coder и Starcoder2.

> [!Note]
> Для запуска модели необходим Ollama.

Установка: 
- <code>pipx install aider-chat</code>, основной пакет, само приложение
- <code>pipx install aider-install</code>, установщик-помощник, созданный разработчиками Aider.

### Запуск модели

```bash
# Переходим в проект
cd ~/путь/к/проекту

# Запускаем, указав модель (для экономии памяти лучше 1.5b или 7b)
aider --model ollama/qwen2.5-coder:1.5b
```
